Pengaruh Teknologi dalam Pembelajaran Modern

Deskripsi

Perkembangan teknologi telah membawa perubahan besar dalam dunia pendidikan. Teknologi memudahkan akses informasi, meningkatkan kreativitas, serta membuat pembelajaran lebih interaktif.

👥 Perkenalan Kelompok

Rey
Graphic Designer

Bertugas membuat desain visual seperti layout, warna, dan tampilan agar website terlihat menarik, estetik, dan nyaman dilihat.

Aufa
Content Writer

Bertugas menyusun dan menulis isi materi, informasi, serta teks yang ditampilkan pada website agar jelas, rapi, dan mudah dipahami.

Ardian
Web Developer

Bertugas membuat dan mengembangkan website menggunakan HTML dan CSS serta menggabungkan semua bagian seperti desain dan konten menjadi satu tampilan utuh.

Ilham
Programmer

Bertugas menambahkan fitur interaktif dan fungsi tambahan pada website agar lebih dinamis dan menarik saat digunakan.

Manfaat Teknologi

  • Memudahkan akses informasi
  • Meningkatkan minat belajar
  • Mendukung belajar mandiri
  • Mempercepat pemahaman
